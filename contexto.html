Menu De Contexto

Uma classe JavaScript que possibilita a criação de menu de contexto para inúmeros elementos.

Usabilidade

Abaixo temos dois exemplos para esta página, clicando com o botão direito do mouse poderá exibi-las.

Não somente poderá navegar usando o mouse como também poderá usar o teclado apertando as telas cima, baixo, direita, esquerda, espaço e enter intuitivamente além de poder usar teclas de atalho para acioná-las sem precisar abrir o menu.

Instanciando o objeto

Para a instância do objeto é necessário que se passe um array de objeto para o construtor. new MenuContexto([{},{},…]);

Os objetos do array devem conter um 'alvo' que seja o id do elemento e o 'dado' que contem as informações do menu de contexto.

Valores aceitos

‘imagem:’ coloca-se o link da imagem onde pode ser qualquer formato suportado pela tag <img />.

‘texto:’ nome da ação a se selecionar.

‘funcao:’ função que deve ser ativada ao clicar.

‘comando:’ essa opção aceita tanto uma letra ou número para o "accesskey" (Alt+) quanto um array de objetos que respeite o formato do menu, para que se abra um submenu, podendo ter imagem, texto, função e comando.

As quatro opções são facultativas e uma vez passado um objeto vazio é gerado um <hr /> criando um separador de categoria.
